## Σύνοψη

Ο πραγματικός φραγμός είναι τα **500 credits/μήνα της The Odds API**· το settle
έτρωγε ~150/μήνα μόνο για τελικά σκορ (commodity data). Νέα εντολή **`fd-settle`**
settle-άρει soccer bets **δωρεάν** μέσω football-data.org (X-Auth-Token, 10 req/min).
Το CLV (closing line Pinnacle) μένει στην The Odds API — δεν αντικαθίσταται.

## Τι έγινε

- `src/football_data_client.mjs` — minimal v4 client· διαβάζει το throttling header
  `X-Requests-Available-Minute` (οδηγία TheSportsDB/FD), δεν διαρρέει το key σε errors.
- `src/football_data_settle.mjs` — map sportKey→competition (WC, PL, SA, BSA, …) +
  `synthesizeFdScoreEvents` (ταίριασμα ανά normalized ονόματα + ημερομηνία, με
  fallback unique-names· skip σε ambiguous/unfinished → fail safe).
- `src/cli.mjs` — νέα εντολή `fd-settle` (settle-άρει μόνο τις FD-καλυπτόμενες soccer
  λίγκες· οι υπόλοιπες μένουν για το `settle`/The Odds API). Loader για
  `football_data_org_key`.
- `scripts/run-paper-settle.ps1` — τρέχει **`fd-settle` ΠΡΩΤΑ** (free), μετά `settle`
  (Odds API για non-soccer) + `mispricing-settle`.
- +7 tests (`football_data.test.mjs`, `cli_fd_settle.test.mjs`). Σύνολο **197**.
