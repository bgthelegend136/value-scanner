σκορ και υπολογίζει πραγματικό ROI.

της Pinnacle — γρήγορο σήμα edge χωρίς να περιμένεις αποτέλεσμα.

- λογική CLV (`src/paper.mjs`) — `7625ecd`
- εντολή `clv` — `d988044`

**Εντολή:** `node src/cli.mjs clv`

## Επιβεβαίωση με πραγματικά δεδομένα (2026-06-24)

```
scan   → 22 matched fixtures, 3 value bets, έγραψε 3 paper bets
clv    → CLV captured: 3, beat rate 100%, average +3.5%
settle → 3 pending, 0 settled (οι αγώνες δεν έχουν παιχτεί ακόμα), ROI: N/A
```

Όλες οι εντολές δουλεύουν end-to-end. Το ROI θα συμπληρωθεί καθώς τελειώνουν οι
αγώνες του Μουντιάλ (τρέξε `settle` τις επόμενες μέρες).

## Καθημερινή ροή χρήσης

1. `node src/cli.mjs scan` — βρες value bets (γράφονται αυτόματα στο ledger).
2. `node src/cli.mjs clv` — **λίγο πριν** την έναρξη, για CLV.
3. `node src/cli.mjs settle` — **αφού** τελειώσουν οι αγώνες, για ROI.
